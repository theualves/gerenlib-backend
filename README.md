# Gerenlib Backend

Este é o servidor de backend (API REST) para o projeto **Gerenlib (Acervus)**, um sistema focado no gerenciamento de empréstimos e devoluções de livros. A API foi desenvolvida em Node.js, utiliza o MongoDB como banco de dados e está hospedada na nuvem através do Render.

## 🔗 Link da API em Produção
A API está pública e pode ser acessada através da URL:  
`https://gerenlib-backend.onrender.com`

---

## 🛠️ Tecnologias Utilizadas

* **[Node.js](https://nodejs.org/):** Ambiente de execução do JavaScript no servidor.
* **[Express](https://expressjs.com/):** Framework minimalista para gerenciamento de rotas e requisições HTTP.
* **[MongoDB Atlas](https://www.mongodb.com/cloud/atlas):** Banco de dados NoSQL hospedado na nuvem.
* **[Mongoose](https://mongoosejs.com/):** Biblioteca ODM para modelagem de dados do MongoDB no Node.js.
* **[Nodemon](https://nodemon.io/):** Ferramenta de desenvolvimento que reinicia o servidor automaticamente a cada alteração no código.
* **[Cors](https://www.npmjs.com/package/cors):** Middleware para permitir que o frontend (Web ou Mobile) consulte a API com segurança.
* **[Dotenv](https://www.npmjs.com/package/dotenv):** Gerenciamento de variáveis de ambiente de forma segura.
